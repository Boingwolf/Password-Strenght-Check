# Password Strenght Check

Este repositório é **apenas** para aprender e evoluir um projeto de verificação de força de palavras-passe em Python.

## 🎯 Objetivo de estudo

O foco é praticar:
- validação de entradas
- critérios de força de password
- feedback ao utilizador
- organização básica de lógica em funções

## 📘 Caminho de aprendizagem sugerido

1. **Começar pelo básico**
   - perceber o que torna uma password fraca ou forte
   - definir regras simples: comprimento, maiúsculas, minúsculas, números e símbolos

2. **Pensar na lógica antes do código**
   - que verificações precisas de fazer?
   - como vais mostrar ao utilizador o resultado?
   - vais dar uma pontuação, níveis, ou ambos?

3. **Melhorar a experiência**
   - mostrar mensagens claras
   - explicar o que falta para a password ficar mais forte
   - evitar respostas demasiado genéricas

4. **Evoluir o projeto**
   - adicionar mais níveis de força
   - detetar padrões comuns e previsíveis
   - testar vários exemplos de passwords
